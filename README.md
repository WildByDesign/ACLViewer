# ACLViewer
ACL Viewer for Windows

- Automatic dark mode / light mode depending on system theme
- Fast ACL/SDDL/ACCESS_MASK parsing
- Displays ACLs for directories and files

# Hotkeys

- F5 to refresh the treeview
